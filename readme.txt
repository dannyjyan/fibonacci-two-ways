1. Save the Python file(s) to your computer
2a. Run this command:
   ```
   python exhaustive.py
   ```

   ```
   python golden_ratio.py
   ```
Follow the instructions and enter two numbers to test equation 3.
The rest of the testing numbers will be printed immediately afterwards.
